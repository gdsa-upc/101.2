Proyecto GDSA Search-Around (101.2)

Somos un grupo de estudiantes de la Universidad Politécnica de Catalunya, que cursamos el grado de Ingenieria de Sistemas Audiovisuales.

Este grupo está formado por los alumnos: Mi Steinbach, Sarai Vizuet, Felipe Christian Cuzcano, Albert Bruna y Joshua Molina.

Nuestro objetivo es enseñar al programa para que consiga los dos objetivos de este proyecto, a través de una base de datos de imágenes (fotos):

      1. Buscador visual

      2. Clasificador de imágenes

Nuestra base de datos estará formada por 900 fotos, de las cuáles 600 fotos serán de varios edificios de Terrassa, y las otras 300 son sobre cualquier otro edifcio. Nuestro grupo se encarga de hacer 50 fotos del ayuntamiento de Terrassa, 50 fotos de la Sociedad General de Electricidad y 50 fotos de otra série de edificios.

En primer lugar, para poder enseñar al programa a que haga una búsqueda visual a través de una imagen que cogemos como referencia, haremos un primer script para que consiga los puntos de interés de dicha imagen. Una vez enseñado eso, el programa debe comparar la imagen de referencia con toda la base de datos y, después, nos debe dar una secuencia de imágenes, las cuales tengan la máxima simlitud a la imagen de referencia.

En segundo lugar, deberemos enseñar al programa para que cuando le demos una imagen de referencia, la clasifique en la base de datos y, una vez hecho esto, debería decir a qué pertenece esta imagen mediante una descripción. Por ejemplo, suponemos que la imagen de referencia es el ayuntamiento de Terrassa; lo que haría el programa sería clasificar esta imagen en nuestra base de datos, y después decirnos que la imagen pertenece al ayuntamiento de Terrassa.
